# Changelog

All notable changes to Bonsai Desk are documented here.

## [0.3.0] - 2026-04-04

- Added Bonsai model switching for `8B`, `4B`, and `1.7B`.
- Added runtime diagnostics in the setup flow and runtime modal.
- Reworked the sidebar into a mobile drawer and improved responsive layout behavior.
- Added frontend logic tests and expanded backend test coverage.
- Improved install docs, transparency copy, and API documentation.

## [0.2.0] - 2026-04-04

- Added the runtime control modal, setup transparency notice, and local asset linking.
- Added persistent UI preferences, runtime presets, and improved error handling.
- Prepared repository docs, CI, and GitHub issue / PR templates.

## [0.1.0] - 2026-04-04

- Initial local chat MVP with FastAPI backend, React frontend, Bonsai runtime management, streaming chat, and SQLite conversation storage.
